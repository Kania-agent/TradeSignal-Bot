# TradeSignal Bot

AI-driven crypto trading signal dashboard powered by MiMo V2.5.

## Features

- Dark trading terminal theme with green/red accent colors
- CSS-only candlestick chart visualization
- Real-time signal cards (BUY/SELL/HOLD)
- Portfolio tracking table with P&L
- Market indicators and trend analysis

## Tech Stack

- Vanilla HTML5, CSS3, JavaScript (ES6+)
- No external dependencies or frameworks

## Usage

Open `index.html` in a browser. The terminal loads with live signal data, candlestick charts, and portfolio information.

## AI Engine

Powered by **MiMo V2.5** — Nous Research's market analysis model.
